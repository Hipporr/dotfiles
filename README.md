# dotfiles

My personal dotfiles for **Hyprland**, **Hyprpaper**, **Waybar**, and **Fastfetch**.  
This repo is mainly for backup and syncing across my systems, but feel free to use or fork it.  

---

## What's Included
- **Hyprland** → WM config  
- **Hyprpaper** → wallpaper configuration its in the hyprland dir
- **Waybar** → my green status bar
- **Fastfetch** → Config for a green fastfetch
